ติดตั้ง Nuxt + Vuetify

คู่มือตั้งต้นแบบ step-by-step

ขั้นตอนหลัก

1 เปิด terminal แล้วไปที่ drive D: 
d:
2 สร้างโปรเจกต์ Nuxt ใหม่ 
npm create nuxt@latest

ตั้งชื่อโปรเจกต์ตามต้องการ เช่น my-app จากนั้นกด Enter ไปเรื่อยๆ จนเสร็จ

3 เปิดโปรเจกต์ใน VS Code 
code d:\my-app -r

เปลี่ยน my-app เป็นชื่อโฟลเดอร์ที่ตั้งไว้

4 แก้ไฟล์ app.vue และย้ายออกมาด้านนอก

ไฟล์ app.vue ต้องอยู่ที่ root ของโปรเจกต์ (ระดับเดียวกับ nuxt.config.ts) ถ้ามันอยู่ในโฟลเดอร์อื่นให้ย้ายออกมาก่อน แล้วแก้เนื้อหาเป็น:

app.vue  ← วางไว้ที่ root โปรเจกต์
<template> <NuxtLayout> <NuxtPage /> </NuxtLayout> </template>
โครงสร้างโฟลเดอร์ที่ถูก:
my-app/
  app.vue ← ตรงนี้
  nuxt.config.ts
  pages/
  public/
5 แก้ไฟล์ nuxt.config.ts

ดูโค้ดได้ที่ Vuetify Manual Setup แล้วคัดลอกค่าตั้งค่าส่วน Nuxt มาใส่

nuxt.config.ts
import vuetify, { transformAssetUrls } from 'vite-plugin-vuetify' export default defineNuxtConfig({ build: { transpile: ['vuetify'], }, modules: [ (_options, nuxt) => { nuxt.hooks.hook('vite:extendConfig', (config) => { // @ts-expect-error config.plugins.push(vuetify({ autoImport: true })) }) }, ], vite: { vue: { template: { transformAssetUrls, }, }, }, })
6 สร้างโฟลเดอร์ pages และไฟล์ index.vue

สร้างโฟลเดอร์ pages/ ที่ root แล้วสร้างไฟล์ index.vue ข้างใน

pages/index.vue
<template> <v-container> <h1>Hello Nuxt + Vuetify</h1> </v-container> </template>
7 ติดตั้ง Vuetify 
npm i -D vuetify
8 ติดตั้ง vite-plugin-vuetify 
npm i -D vite-plugin-vuetify

คำแนะนำพิเศษ

★ โฟลเดอร์ img สำหรับรูปภาพ
สร้างโฟลเดอร์ public/img/ สำหรับเก็บรูปภาพทั้งหมด ไฟล์ที่อยู่ใน public/ จะถูก serve ตรงๆ โดยไม่ต้อง import เช่น ถ้ามีไฟล์ public/img/banner.jpg ก็เรียกใช้ใน template ได้เลยว่า: 
<img src="/img/banner.jpg" />
★ Generate ไฟล์สำหรับ deploy (Static Site)
เมื่อพร้อม deploy ให้รันคำสั่ง: 
npx nuxi generate
ไฟล์ทั้งหมดจะถูก build ออกมาที่โฟลเดอร์ .output/public/ นำเนื้อหาใน .output/public/ ไปอัปโหลดขึ้น GitHub Pages หรือ Netlify ได้เลย (ไม่ต้องอัป node_modules หรือ source code)
