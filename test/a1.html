ติดตั้ง Nuxt + Vuetify

คู่มือตั้งต้นแบบ step-by-step

ขั้นตอนหลัก

1 เปิด terminal แล้วไปที่ drive D:

d:

2 สร้างโปรเจกต์ Nuxt ใหม่

npm create nuxt@latest

ตั้งชื่อโปรเจกต์ เช่น my-app แล้วกด Enter ไปเรื่อยๆ จนเสร็จ

3 เข้าโฟลเดอร์ แล้วติดตั้ง dependencies

cd my-app
npm install

4 เปิดโปรเจกต์ใน VS Code

code d:\my-app -r

เปลี่ยน my-app เป็นชื่อโฟลเดอร์ที่ตั้งไว้

5 ติดตั้ง Vuetify และ plugin

npm i -D vuetify vite-plugin-vuetify @mdi/font

@mdi/font คือชุดไอคอนที่ Vuetify ใช้เป็นค่าเริ่มต้น

6 แก้ไฟล์ app.vue (วางไว้ที่ root)

ไฟล์ app.vue ต้องอยู่ระดับเดียวกับ nuxt.config.ts

app.vue

<template>
  <NuxtLayout>
    <NuxtPage />
  </NuxtLayout>
</template>
โครงสร้างที่ถูก: my-app/ → app.vue, nuxt.config.ts, pages/, plugins/, public/

7 แก้ไฟล์ nuxt.config.ts

อ้างอิง Vuetify Manual Setup

nuxt.config.ts

import vuetify, { transformAssetUrls } from 'vite-plugin-vuetify'

export default defineNuxtConfig({
  build: { transpile: ['vuetify'] },
  css: ['vuetify/styles', '@mdi/font/css/materialdesignicons.css'],
  modules: [
    (_options, nuxt) => {
      nuxt.hooks.hook('vite:extendConfig', (config) => {
        // @ts-expect-error
        config.plugins.push(vuetify({ autoImport: true }))
      })
    },
  ],
  vite: {
    vue: { template: { transformAssetUrls } },
  },
})

8 สร้างไฟล์ plugins/vuetify.ts

ขั้นนี้สำคัญ — ถ้าไม่มี Vuetify จะไม่ถูก register ทำให้ <v-...> ใช้ไม่ได้

plugins/vuetify.ts

import { createVuetify } from 'vuetify'
import * as components from 'vuetify/components'
import * as directives from 'vuetify/directives'

export default defineNuxtPlugin((app) => {
  const vuetify = createVuetify({ components, directives })
  app.vueApp.use(vuetify)
})

9 สร้าง pages/index.vue

สร้างโฟลเดอร์ pages/ ที่ root แล้วสร้างไฟล์ข้างใน

pages/index.vue

<template>
  <v-container>
    <h1>Hello Nuxt + Vuetify</h1>
    <v-btn color="primary">ปุ่มทดสอบ</v-btn>
  </v-container>
</template>

10 รันเพื่อทดสอบ

npm run dev

เปิด http://localhost:3000 ถ้าเห็นปุ่มสีฟ้า แสดงว่า Vuetify ทำงานแล้ว

คำแนะนำพิเศษ

★ โฟลเดอร์ img สำหรับรูปภาพ

สร้าง public/img/ เก็บรูป — ไฟล์ใน public/ เรียกใช้ได้ตรงๆ โดยไม่ต้อง import

<img src="/img/banner.jpg" />

★ Generate ไฟล์สำหรับ deploy (Static)

เมื่อพร้อม deploy ให้รัน:

npx nuxi generate

ไฟล์จะ build ออกที่ .output/public/ นำเนื้อหาในนั้นไปอัปขึ้น GitHub Pages หรือ Netlify ได้เลย (ไม่ต้องอัป node_modules หรือ source code)
