ติดตั้ง Vuetify บน Nuxt (วิธี Manual)

สคริปต์อัตโนมัติสำหรับ Windows ติดตั้ง Vuetify, ตั้งค่า plugin และแก้ nuxt.config ให้ในคลิกเดียว

อ่านก่อนรัน

ต้องมีก่อน
  • เครื่อง Windows (สคริปต์ใช้ PowerShell / ไฟล์ .bat)
  • ติดตั้ง Node.js และ npm เรียบร้อยแล้ว
  • มีโปรเจกต์ Nuxt อยู่แล้ว (มีไฟล์ nuxt.config.ts หรือ .js)
  • วางไฟล์ .bat และ .ps1 ไว้ โฟลเดอร์เดียวกัน

สคริปต์จะสำรองไฟล์เดิมเป็น .bak ให้อัตโนมัติก่อนเขียนทับ ถ้า config เดิมมีของอื่นอยู่ ให้เปิดไฟล์ .bak มา merge กลับเอง

วิธีใช้งาน

  1. โหลดไฟล์ทั้งสองด้านล่าง แล้ววางไว้ในโฟลเดอร์เดียวกัน
  2. ดับเบิลคลิกไฟล์ add-vuetify-manual.bat
  3. เมื่อถูกถาม ให้วางพาทโฟลเดอร์โปรเจกต์ Nuxt เช่น D:\my-app แล้วกด Enter 
    วางพาทโฟลเดอร์โปรเจกต์ Nuxt (เช่น D:\my-app)
  4. รอจนขึ้นว่า เสร็จแล้ว! จากนั้นเข้าโปรเจกต์แล้วรัน 
    npm run dev

ดาวน์โหลดไฟล์

add-vuetify-manual.bat ไฟล์ที่ต้องดับเบิลคลิกเพื่อเริ่มทำงาน เรียก PowerShell ให้อัตโนมัติ ดาวน์โหลด .bat
setup-vuetify-manual.ps1 สคริปต์หลักที่ติดตั้ง package และตั้งค่าไฟล์ทั้งหมด (ต้องอยู่คู่กับ .bat) ดาวน์โหลด .ps1

สคริปต์ทำอะไรให้บ้าง

  • ถอน vuetify-nuxt-module ออกถ้ามี (เพราะชนกับวิธี manual)
  • ติดตั้ง vuetify, vite-plugin-vuetify, @mdi/font
  • เขียน nuxt.config ใหม่ (สำรองของเดิมเป็น .bak)
  • สร้าง plugins/vuetify.ts
  • ครอบ <NuxtPage /> ด้วย <v-app> ใน app.vue
  • สร้าง pages/index.vue และ public/img ถ้ายังไม่มี
